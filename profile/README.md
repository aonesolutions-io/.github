# AONE

Building India's next generation mobility platform.

## About

AONE builds mobility software for customers, drivers, fleet operators, and internal teams.

## Products

- Customer App
- Driver App
- Fleet App
- Admin Portal
- Public Website

## Technology

- React Native
- Expo
- Node.js
- TypeScript
- PostgreSQL
- Redis
- Google Cloud Platform

## Engineering

- Documentation-first delivery
- Small pull requests
- Automated testing
- Security by design
